# 🚀 Microsoft-Level EdTech Learning Platform

AI Powered Full Stack EdTech Ecosystem  
Built with Django, React.js, MySQL, Redis, Docker, Celery, Three.js, and AI Recommendation Systems.

---

# 🌍 Project Vision

This platform is designed to simulate a real-world enterprise-grade learning ecosystem similar to platforms used by Microsoft, Coursera, Udemy, and LinkedIn Learning.

The system includes:

- AI-powered recommendations
- Real-time chat classrooms
- Student analytics dashboards
- OAuth2 authentication
- Automated PDF certificate generation
- Payment gateway architecture
- Community discussion system
- Dockerized deployment
- Celery async task processing
- Redis caching and websocket support
- Modular scalable backend architecture

# ⚡ Installation Guide

# 1️⃣ Clone Repository

```bash
git clone https://github.com/dinesh-gaikwad/EdTech-Learning-Platform.git

cd EdTech-Learning-Platform
```

---

# 2️⃣ Run Using Docker

```bash
docker-compose up --build
```

---

# 3️⃣ Frontend URL

```bash
http://localhost:5173
```

---

# 4️⃣ Backend URL

```bash
http://localhost:8000
```

---

# 5️⃣ Django Admin

```bash
http://localhost:8000/admin
```
